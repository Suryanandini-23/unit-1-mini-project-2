# Student Information Portal

## Project Title

Student Information Portal

## Problem Statement

Create a website for managing and presenting student academic and personal information.

## Project Description

The Student Information Portal is a simple HTML-based website developed to display and manage student details. The portal provides information about students, including their personal details and academic records. Users can navigate between pages using hyperlinks and view student information in a structured format.

## Objectives

* To create a multi-page website using HTML.
* To present student personal information.
* To present student academic information.
* To demonstrate the use of HTML elements such as tables, lists, images, hyperlinks, and forms.
* To provide easy navigation between web pages.

## Features

* Home Page
* Student Information Page
* Navigation Menu
* Student Personal Details
* Student Academic Details
* Tables for Data Presentation
* Images and Lists
* User-Friendly Interface

## Technologies Used

* HTML5

## Pages Description

### Home Page (index.html)

* Welcome message

### Student Information Page (student.html)

* Student records
* Personal details
* Academic details
* Tables for displaying information



## Expected Output

The website allows users to navigate from the Home Page to the Student Information Page and view personal and academic details of multiple students.
